# Tartas Anamil

🇬🇧 [English](#english) · 🇪🇸 [Español](#español)

---

## English

Landing page for a local bakery specializing in custom handmade cakes in Madrid. Customers order directly via WhatsApp. The client specifically requested a girly, playful visual style — reflected in the pixel art aesthetic, pastel tones, animated drips and decorative characters.

**Tech:** HTML · CSS · JavaScript

**Features:**
- Pixel art aesthetic with animated hero, drip effects and decorative characters
- Infinite marquee banner with highlights
- Cake menu section with available options
- Photo gallery of past designs
- 3-step ordering process (WhatsApp → details → pick up)
- Floating WhatsApp button for instant contact
- Scroll reveal animations
- Fully responsive

**Usage:**
Open `index.html` in your browser.

## Español

Landing page para una pastelería local especializada en tartas artesanales por encargo en Madrid. Los pedidos se gestionan directamente por WhatsApp. El cliente pidió expresamente un estilo visual muy femenino y juguetón — reflejado en la estética pixel art, tonos pastel, goteos animados y personajes decorativos.

**Tecnologías:** HTML · CSS · JavaScript

**Funcionalidades:**
- Estética pixel art con hero animado, efecto goteo y personajes decorativos
- Banner marquee infinito con highlights
- Sección de menú con las tartas disponibles
- Galería de fotos con diseños anteriores
- Proceso de pedido en 3 pasos (WhatsApp → detalles → recogida)
- Botón flotante de WhatsApp para contacto directo
- Animaciones de scroll reveal
- Totalmente responsive

**Uso:**
Abre `index.html` en tu navegador.
